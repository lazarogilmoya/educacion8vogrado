El diseño web se refiere al proceso de planificar y crear la estructura, apariencia y funcionalidad de un sitio web. Esto incluye el diseño de la interfaz, la disposición de los elementos, la selección de colores, fuentes y gráficos, y la adaptación a diferentes dispositivos y tamaños de pantalla. Un buen diseño web asegura que un sitio sea visualmente atractivo, fácil de navegar y accesible para todos los usuarios. Además, debe optimizarse para ofrecer una experiencia de usuario fluida y eficiente

Video 2

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. HTML define la estructura de un documento web mediante el uso de etiquetas y atributos que indican cómo se debe mostrar el contenido. Permite la inclusión de texto, imágenes, enlaces y otros elementos multimedia. Un documento HTML está compuesto por una serie de elementos anidados que describen la organización del contenido y facilitan la presentación en navegadores web. Su propósito es estructurar la información de manera que sea comprensible y accesible tanto para los usuarios como para los motores de búsqueda.

La programación para niños se centra en enseñar los conceptos fundamentales de codificación de manera divertida y accesible. Incluye conceptos básicos como variables, bucles, y condicionales, que se utilizan para crear instrucciones que una computadora puede seguir. Las actividades suelen involucrar juegos y proyectos creativos que permiten a los niños construir sus propios programas y ver los resultados de inmediato. Este enfoque fomenta habilidades de resolución de problemas, lógica y pensamiento computacional desde una edad temprana.
