Glosario

API (Application Programming Interface): Conjunto de protocolos y herramientas para construir software y aplicaciones.
Back-end: Parte del desarrollo web que se encarga de la lógica del servidor, bases de datos y la integración del sistema.
CSS (Cascading Style Sheets): Lenguaje de diseño utilizado para describir la presentación de un documento HTML.
DOM (Document Object Model): Modelo de programación que permite el acceso y la manipulación del contenido y estructura de los documentos HTML y XML.
Element: Cada etiqueta en HTML que define una estructura o contenido.
Framework: Conjunto de herramientas y bibliotecas que facilitan el desarrollo de aplicaciones y sistemas.
GUI (Graphical User Interface): Interfaz gráfica de usuario que permite la interacción con el software a través de elementos visuales.
HTML (HyperText Markup Language): El lenguaje de marcado estándar para crear páginas web.
IDE (Integrated Development Environment): Entorno de desarrollo integrado que facilita la programación con herramientas como editores de código, depuradores, y compiladores.
JavaScript: Lenguaje de programación utilizado principalmente para crear contenido interactivo en páginas web.
Keyword: Palabra clave utilizada en lenguajes de programación y motores de búsqueda.
Library: Conjunto de funciones y procedimientos reutilizables en programación.
Multimedia: Contenido que utiliza una combinación de diferentes formas de contenido como texto, audio, imágenes, animaciones, video e interactividad.
Node.js: Entorno de ejecución para JavaScript en el lado del servidor.
Open Source: Software de código abierto que se puede utilizar, modificar y distribuir libremente.
Protocol: Conjunto de reglas que determinan cómo se transmiten los datos a través de una red.
Query: Consulta realizada a una base de datos para recuperar información específica.
Responsive Design: Diseño web que se adapta automáticamente a diferentes tamaños de pantalla y dispositivos.
SEO (Search Engine Optimization): Proceso de mejorar la visibilidad de un sitio web en los motores de búsqueda.
Tag: Marcador en HTML que define elementos en un documento, por ejemplo,
.
URL (Uniform Resource Locator): Dirección de una página web.
Viewport: Área visible de una página web en una pantalla o dispositivo.
Web Server: Computadora que almacena y entrega páginas web a través de Internet.
XML (eXtensible Markup Language): Lenguaje de marcado utilizado para almacenar y transportar datos.
YAML (YAML Ain't Markup Language): Lenguaje de serialización de datos fácil de leer, utilizado para la configuración de aplicaciones.
Z-index: Propiedad en CSS que especifica el orden de apilamiento de los elementos en una página web.
