Inserción de Enlaces, Hipervínculos o Links

En HTML, los enlaces o hipervínculos permiten conectar una página web con otras páginas, documentos o recursos en la web. Los enlaces se crean utilizando la etiqueta ``. A continuación, se explica cómo insertar enlaces y algunos atributos útiles:

1. Crear un Enlace Básico

Para crear un enlace básico, se utiliza la etiqueta `` con el atributo `href`, que especifica la URL de destino:

<a href="https://www.ejemplo.com">Visita Ejemplo</a>

En este caso, al hacer clic en "Visita Ejemplo", el usuario será redirigido a www.ejemplo.com.

2. Enlaces a Páginas Internas

Para enlazar a otras páginas dentro del mismo sitio web, se utiliza la ruta relativa:

<a href="pagina_interna.html">Ir a Página Interna</a>

3. Abrir Enlaces en una Nueva Ventana

Para que un enlace se abra en una nueva ventana o pestaña, se usa el atributo `target` con el valor `_blank`:

<a href="https://www.ejemplo.com" target="_blank">Abrir en nueva ventana</a>

4. Enlaces a Correo Electrónico

Para crear un enlace que abra el cliente de correo electrónico del usuario, se utiliza el protocolo `mailto:`:

<a href="mailto:correo@ejemplo.com">Enviar correo</a>

5. Enlaces a Teléfonos

Para crear un enlace que permita realizar una llamada telefónica desde dispositivos compatibles, se usa el protocolo `tel:`:

<a href="tel:+123456789">Llamar al +123456789</a>

6. Enlaces Ancla

Los enlaces ancla permiten enlazar a una sección específica dentro de la misma página. Se usa el atributo `id` en el elemento de destino y el carácter `#` en el enlace:

<a href="#seccion1">Ir a Sección 1</a>
<h2 id="seccion1">Sección 1</h2>

7. Enlaces con Imagen

También se pueden usar imágenes como enlaces. La etiqueta `` se coloca dentro de la etiqueta ``:

<a href="https://www.ejemplo.com">
    <img src="imagen.jpg" alt="Descripción de la imagen">
</a>

8. Enlaces de Descarga

Para proporcionar un enlace de descarga a un archivo, se utiliza el atributo `download`:

<a href="archivo.pdf" download="nombre_archivo.pdf">Descargar PDF</a>
