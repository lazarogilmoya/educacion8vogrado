Tablas en HTML

Las tablas en HTML se utilizan para presentar datos tabulares. Se componen de filas (<tr>), celdas de encabezado (<th>) y celdas de datos (<td>). A continuación, se muestra un ejemplo de cómo crear y estilizar una tabla:

Ejemplo de Tabla

Maria García 30 Barcelona
Pedro Martínez 28 Valencia

Explicación de la Tabla

  • <table>: Define una tabla.
  • <thead>: Agrupa los encabezados de la tabla.
  • <tbody>: Agrupa el cuerpo de la tabla.
  • <tr>: Define una fila en la tabla.
  • <th>: Define una celda de encabezado (se muestra en negrita y centrada por defecto).
  • <td>: Define una celda de datos.

Estilización de Tablas

Las tablas se pueden estilizar con CSS para mejorar su apariencia. A continuación, se muestra un ejemplo básico de estilización:

table {
    width: 100%;
    border-collapse: collapse;
    margin: 20px 0;
    font-size: 18px;
    text-align: left;
}

th, td {
    padding: 12px;
    border: 1px solid #ddd;
}

th {
    background-color: #f4f4f4;
    color: #333;
}

tr:nth-child(even) {
    background-color: #f9f9f9;
}

tr:hover {
    background-color: #f1f1f1;
}

Este código CSS aplica un estilo simple a las tablas, incluyendo bordes, espaciado y colores alternos para las filas.
