Inserción de Imágenes

En HTML, puedes insertar imágenes en tus páginas web utilizando la etiqueta <img>. La etiqueta <img> no tiene una etiqueta de cierre y se usa junto con los atributos src y alt.

1. Estructura Básica

Para insertar una imagen, utiliza el siguiente código HTML:

<img src="ruta/a/tu/imagen.jpg" alt="Descripción de la imagen">

Donde src especifica la ruta del archivo de imagen y alt proporciona una descripción alternativa para la imagen.

2. Atributos de la Etiqueta <img>

  • src: Especifica la ubicación del archivo de imagen.
  • alt: Proporciona un texto alternativo que describe la imagen para usuarios que no pueden verla.
  • width: Define el ancho de la imagen en píxeles.
  • height: Define la altura de la imagen en píxeles.
  • title: Muestra un texto emergente cuando el usuario pasa el cursor sobre la imagen.

3. Ejemplos de Inserción de Imágenes

Imagen con Texto Alternativo

Este es un ejemplo de una imagen con una descripción alternativa:

<img src="assets/example1.jpg" alt="Una imagen de ejemplo">

Imagen con Tamaño Definido

También puedes definir el tamaño de la imagen:

<img src="assets/example2.jpg" alt="Una imagen con tamaño definido" width="300" height="200">

Imagen con Título

Agrega un título que aparece al pasar el cursor:

<img src="assets/example3.jpg" alt="Imagen con título" title="Este es un título de la imagen">

4. Buenas Prácticas

  • Asegúrate de que las imágenes tengan un texto alternativo descriptivo para accesibilidad.
  • Optimiza las imágenes para la web para mejorar el tiempo de carga.
  • Utiliza rutas relativas o absolutas adecuadas para la ubicación de tus imágenes.
  • Considera usar formatos modernos como WebP para mejorar la calidad y reducir el tamaño del archivo.
